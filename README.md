# Firebase IoT
Internet Of Things (IoT) proyek dengan data realtime firebase. Mengatur relay, mendapatkan informasi udara (%), suhu (C) & ppm (Zat)

# Item
```
  1. Wemos D1 Mini ESP8266
  2. RelayKeys
  3. LCDI2C
  4. TDS
  5. DHT11
```

# Pasang
```
  Flash
  Buka esp8266flasher.exe
  ----------------------------
  Advanced
  115200
  4Mb
  40Mhz
  DOUT
  ----------------------------
  Config
  Dir Your_Local_Pc/Desktop/wemos_d1_mini_esp8266_lucknet_iot.bin    |    0x00000
  ----------------------------
  Operation
  Your Port Ex.COM1 ---> Flash(F)
```
